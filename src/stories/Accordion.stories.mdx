# Accordion

O componente `Accordion` permite que uma área seja encolhida e/ou expandida. Deve ser utilizado para agrupar ou ocultar regiões complexas, mantendo a página limpa. A Sua utilização baseia-se na importação e utilização de 4 componentes. O `AccordionsWrapper` deve ser o container em volta de todos os Accordions, cada item consiste (Nesta Ordem) de um componente `Accordion` que você deve colocar como filhos dele o componente `AccordionHeader` que define o seu título e o conteúdo (children) do Accordion sendo definido no componente `AccordionDetail`.

# Expanded

É possível definir o estado inicial de aberto/fechado do accordion, para isto basta utilizar a prop `expanded` com o valor desejado, por padrão é `false`. O accordion controla o seu estado de aberto/fechado, se você deseja controlar isso externamete é necessário que também utilize a prop `disabled`, desta forma o componente não irá mais mudar o seu estado, ficando a seu cargo passar o valor através da prop `expanded`

# Disabled
