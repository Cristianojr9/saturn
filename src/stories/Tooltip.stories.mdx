# Variant

O `Tooltip` pode receber a prop `variant`, para mostrar informações fundamentais para o usuário. Essa prop também é utilizada para identificar a cor do componente, cujo nome referencia a sua necessidade.

      <br/>
    </ThemeProvider>
  </Story>
</Canvas>

# Small

O `Tooltip` pode também receber a prop `small`, possuindo um tamanho reduzido, utilizado geralmente em informações menores, e/ou de palavra única.
