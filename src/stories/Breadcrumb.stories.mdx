# Breadcrumb

O component de Breadcrumb permite o usuário à enxergar o caminho percorrido além de navegar atravéz do mesmo.

A variante padrão é `responsive`, ou seja, no desktop conta com sua visualização completa e no mobile irá ser reduzido, apontando para a página anterior.

# Condensed

A variante `condensed` irá sempre apontar para à página anterior, tanto em resoluções maiores quanto menores.

<Canvas>
  <Story
    name='Condensed'
    parameters={{
      design: {
        type: 'figma',
        url: 'https://www.figma.com/file/O3bKxIcsj2rc1FNIRclJyT/Saturn-System?node-id=253%3A913'
      }
    }}
  >
    <SaturnProvider>
      <Breadcrumb variant='condensed' path={[{ label: 'Página anterior', link: '/' }]} />
    </SaturnProvider>
  </Story>
</Canvas>

# Expanded

A variante `expanded` segue o mesmo caminho da anterior. Porém, nesse caso, irá mostrar sempre todo o caminho percorrido.
