# Select

O componente de `Select` é utilizado para que o usuário possa selecionar uma ou mais opções da aplicação.

<Preview>
  <Story
    name='Base'
    parameters={{
      design: {
        type: 'figma',

      }
    }}
  >
    <ThemeProvider>
      <Select
        label='Label'
        options={optionsSelect}
        optionLabel='description'
        optionValue='name'
        caption='Caption'
      />
    </ThemeProvider>
  </Story>
</Preview>

# Error

Passando a prop de `error` para o `Select`, o usuário pode receber um feedback visível que existe algum ponto a ser alterado na option em questão.

<Preview>
  <Story
    name='Error'
    parameters={{
