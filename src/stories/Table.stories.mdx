# Tabela

- O componente `Table` permite que sejam criadas tabelas estilizadas para apresentação de dados na aplicação. É possível passar a prop `hideExternalBorder={true}` para utilizar a variação da tabela sem a borda externa.
  Para compor uma tabela utilizando o Saturn você deverá seguir os padrões de [sintaxe nativos do html](https://developer.mozilla.org/pt-BR/docs/Web/HTML/Element/table).

- O componente `TableContent` pode funcionar como as tags `thead` ou `tbody` do HTML, que definem respectivamente o cabeçalho e o corpo da tabela.
- `TableRow` define uma linha da tabela, equivale à tag `tr` do HTML.
- Os `TableData` ficam dentro das `TableRow` e são as células de conteúdo equivalente às tags `td` e `th` do HTML.

## Tabela `regular`

O tipo padrão de tabela é a `regular`, que se diferencia do outro tipo disponível devido à borda externa que a compõe.
No cabeçalho da tabela desse exemplo, os dados dos `TableData` estão sendo definidos com o método `map` dentro de um array. Existe também a opção de definir um por um, como mostrado no código da tabela seguinte.

## Tabela `hideExternalBorder`

A tabela do tipo `hideExternalBorder` possui as mesmas propriedades da `regular`, exceto as bordas externas.

<Canvas>
  <Story
    name='hideExternalBorder'
    parameters={{
      design: {
        type: 'figma',

    }}
  >
    <SaturnProvider>
      <Table hideExternalBorder={true}>
        <TableContent as='thead'>
          <TableData>
